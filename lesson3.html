Mobile App Development – Flutter

Lesson 3: Android vs iOS

  • Android runs on many devices
  • iOS runs only on Apple devices
  • Flutter supports both

Flutter Example


import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Android vs iOS')),
        body: Center(child: Text('Flutter works on both')),
      ),
    );
  }
}
